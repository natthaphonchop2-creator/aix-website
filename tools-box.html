Skill Set สำหรับสมาชิก

เริ่มจากทักษะหลักที่ต้องใช้ซ้ำในทุกคลาส แล้วค่อยต่อยอดเป็นงานจริง

  • Prompt Structureตั้งโจทย์, role, context, output และเกณฑ์ตรวจผลลัพธ์
  • AI Agent Thinkingแตกงานเป็น goal, task, memory, tool และ checkpoint
  • Automation Mappingวาง process ก่อนเชื่อมเครื่องมือจริง

Ebook และคู่มืออ่านประกอบ

พื้นที่รวมคู่มือ PDF, reading notes และ playbook ที่ใช้ควบคู่กับคอร์ส

  • AI Business Playbookโครงสร้างคิดแบบผู้ประกอบการเมื่อต้องนำ AI เข้าธุรกิจ
  • Automation Checklistเช็คความพร้อมก่อนเริ่มทำระบบ automation
  • Research Notesสรุปคำสั่งและรูปแบบการวิจัยด้วย AI

Prompt Pack

ชุด prompt ตั้งต้นสำหรับงานที่ใช้บ่อย และปรับต่อได้ตามธุรกิจของคุณ

  • Content Systemวาง pillar, brief, script และ repurpose content
  • Business Analysisวิเคราะห์ลูกค้า ข้อเสนอ และคู่แข่ง
  • Prompt QAตรวจ prompt ก่อนส่งให้ AI ทำงานจริง

Workflow Blueprint

แบบร่าง workflow สำหรับวางระบบก่อนใช้ n8n, Make, Zapier หรือ custom agent

  • Agent Workflowโครงสร้าง input, decision, action และ output review
  • Lead Automationรับ lead, จัดกลุ่ม, ส่งต่อ และติดตามผล
  • Content Pipelineจาก research ไปจนถึง production และ publishing

Templates

ไฟล์ตั้งต้นสำหรับนำไปทำงานจริง ลดเวลาสร้างระบบเอกสารตั้งแต่ศูนย์

  • Prompt Review Sheetตารางตรวจคุณภาพ prompt สำหรับทีม
  • AI SOP Templateโครง SOP สำหรับงานที่ใช้ AI ช่วยประจำ
  • Project Briefฟอร์มเก็บ context ก่อนเริ่มงาน AI

AI Tool Stack

แนะนำกลุ่มเครื่องมือที่ควรมี พร้อมตำแหน่งการใช้งานในระบบงานจริง

  • Research & Writingเครื่องมือสำหรับวิเคราะห์ข้อมูลและสร้าง draft
  • Automation & Databaseเครื่องมือเชื่อม workflow, form, sheet และ CRM
  • Media & Creativeเครื่องมือภาพ วิดีโอ เสียง และงานนำเสนอ
